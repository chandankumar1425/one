How javascript is executed?.

Javascript is executed in two phases 
1. Memory Component (variable Component) Memory Creation Phase
2. Code Component (Thread of execution) Code Execution phase

Memory Component contains the variables and functions as key:value pairs

Code Component is where the whole JS code is being executed

What is Hoisting in Javascript?.
JavaScript Hoisting refers to the process whereby the interpreter appears to move the declaration of functions, variables or classes to the top of their scope, prior to execution of the code.

Hoisting allows functions to be safely used in code before they are declared.

What is a temporal dead zone?.

The time since when this let variable is hoisted and till theis let variable is initalized a value.

What is closures?.

A function combined with it's lexical scope.

What is a Promise?.
The Promise object represents the eventual completion (or failure) of an asynchronous operation and its resulting value.
